# Roadmap — speex-kit

## ✅ v0.8.0 (Current)
- NDArray with broadcasting, slicing, matmul, axis reductions, concatenation
- Functional toolkit: curry, pipe, memoization, combinators, call-control
- 18 new object/dict operations (pickBy, mapKeys, invert, diff, deepFreeze, etc.)
- 18 new validators (IP, UUID, password, credit card, slug, port, etc.)
- Case conversion (upperFirst, startCase, lines, chars), escapeRegExp
- debounceAsync, detect, mapSeries, eachSeries
- 8 new type guards (isPlainObject, isError, isTypedArray, isArguments, etc.)
- Color: rgbToHsl, hslToRgb, saturate, desaturate, adjustHue, rgba
- IO: safeJsonStringify, envArray

## 🔜 v0.9.0
- Reactive signals (signal/computed/effect) — framework-agnostic
- ML utilities (cosineSimilarity, k-means, confusion matrix)
- AES-GCM encrypt/decrypt (subtle-crypto wrapper)
- Streaming CSV/JSONL parser

## 💡 v1.0.0
- API freeze — no breaking changes
- TypeDoc API documentation
- Benchmark suite vs lodash, date-fns, dayjs, validator.js
- VS Code extension with snippets
